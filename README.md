# FINGERPRINT BASED BIOMETRIC ATTENDANCE SYSTEM WITH GOOGLE SHEET
The Google Sheet Attendance Tracker using ESP8266 and a Biometric Sensor is a smart attendance management system aimed at automating the traditional attendance process. This project utilizes a fingerprint sensor for biometric authentication and an ESP8266 Wi-Fi module to connect to the internet. Once a fingerprint is recognized, the data is sent and recorded directly into a Google Sheet in real time. This system ensures accurate attendance tracking, reduces manual errors, and provides easy access to attendance records from anywhere. It is ideal for educational institutions, offices, or any organization where attendance tracking is necessary.
